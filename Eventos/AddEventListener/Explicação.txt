Possível adicionar eventos tanto pelo arquivo HTML, quanto ao JS. Possível fazer qualquer tipo de evento. Para cada evento, possível chamar uma função. AddEventListener irá servir para ''escutar'', tratar o objeto que o elemento quer para um evento. Dentro de (), será colocado a função no qual será o evento. Ela será colocada após indicar oq será feito: c1.addEventListener("click",msg). Possível ser um arrow function como a função, e editar dentro deste mesmo arrow function o evento.
Ao executar um evento, será capaz de capturar o elemento que disparou o evento. Dentro do console do navegador, será capaz de buscar o target no qual foi o causador do evento. Após descobrir o target, será possível fazer ainda mais modificações.
